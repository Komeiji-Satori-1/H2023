## 算法整理（含补充理由后的版本）

### 题目前提保证
- A和B频率都在20KHz到100Khz之间，且A频率小于B频率，且频率一定是5Khz的整数倍
- top-2峰值一定分属两个不同信号（两个信号峰峰值均为1V保证，三角波基频≈0.811V远大于其3次谐波≈0.090V）
- B的3次/5次谐波**不可能**污染A的3次谐波位置（否则 $f_B \leq f_A$，矛盾）
- 因此A的3次谐波位置的唯一可能污染源只有**B的基频**
- 而A的五次谐波可能被B基波污染。如A为20Khz，5*fa=100Khz，此时B为100Khz就会污染。
- 当然A的五次谐波也有可能被B的三次谐波污染，比如说A为30Khz，5*fa=150KHz，只需fb=50Khz即可。
- 但是实际上我们一般只判断三次谐波就可以解决大部分问题了
---

### 完整流程

```
Step 1: 找top-2基频峰值 → A->bin=低频A, B->bin=高频B（已完成）
Step 2: 预取 A的3次(3fA)、5次(5fA) 位置幅值


Step 3: 分支
  Case <1>: A的3次≈0 → 跳 Step 5
  Case <2>: A的3次有幅值 → 跳 Step 4
  Case <3>: A的3次≈0 但 5次有幅值。说明A的五次和B基频或三次谐波重叠，无论哪种情况A都可以判定为正弦
            → A=正弦（三角波3次必强于5次，3次≈0说明A无谐波）
            → B类型待补充（查B的3次谐波 3×fB）

Step 4: A的3次有幅值，A可能干扰B
  Branch <2> 无重叠（3fA ≠ fB）:
    → 3次幅值来自A自身 → A=三角波
    branch<2.1>5fA=3fB
      →用5fB判断B波形
      →B的五次谐波约等于0即为正弦，反正即为三角。
    branch<2.2>5fA≠3fB
      →用3fB判断B波形
      →B的三次谐波约等于0即为正弦，反正即为三角。
  Branch <1> 有重叠（3fA = fB，此时5fA一定不等于3fB）:
    用5fA判断A波形
    branch<1.1>A的五次谐波约为0
      A为正弦
      →A不会干扰B
      →用3fB判断B波形
      →B的三次谐波约等于0即为正弦，反正即为三角
    branch<1.2>A的五次谐波不为0
      A为三角
      →由于3fA = fB，所以5fA一定不等于3fB
      →用3fB判断B波形
      →B的三次谐波约等于0即为正弦，反正即为三角

Step 5: A的3次≈0
  → A=正弦 ✓
  → A不会干扰B
  → 查B的3次谐波（3fB，无污染）:
      - 比值≈1/9 → B=三角波 ✓
      - 比值≈0   → B=正弦   ✓
```
